# FASE 3 — MENSAJEROS Y OPERACIÓN

## 🎯 Objetivo
Gestionar disponibilidad y jornada de mensajeros.

## 📚 Fuente de verdad
SRS:
- Estados del mensajero
- HU-09, HU-10, HU-11
- RN-01

## 🧩 Alcance

### Funcionalidades
- Iniciar jornada
- Aceptar pedidos
- Entregar pedidos
- Finalizar jornada
- Consultar mensajeros activos

## 🗃️ Entidades
- mensajero
- servicios

## 🔌 Endpoints

- POST /mensajeros/start
- POST /mensajeros/end
- GET /mensajeros/activos
- Incluir endpoints faltantes para los cambios de estado de los servicios

## ⚙️ Reglas clave

- No puede finalizar jornada con servicios activos
- Solo mensajeros DISPONIBLES reciben servicios
- Solo el mensajero asignado puede ver el servicio


## ✅ Definition of Done

- Mensajero puede iniciar jornada
- Mensajero puede aceptar servicios
- Mensajero puede entregar servicios
- Mensajero puede finalizar jornada (si cumple reglas)
- Sistema valida servicios activos

## 🧠 Estado final del sistema

- Operación controlada en tiempo real
- Mensajeros gestionados correctamente
